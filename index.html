Sesión 1

Realizamos presentación individual, asi mismo descargamos una plantilla HTML para una página web donde editamos las letras e imágenes siguiendo esta ruta: -Click derecho en la palabra a editar, seleccionar la palabra "Inspeccionar", posteriormente en el lateral derecho aparecerá el codigo que se copiara y pegaremos en la aplicación Visual Estudio Code con ayuda del control F, luego realizamos los cambios necesarios y guardamos.

Sesión 2

En la sesión 2 realizamos el duplicado de ciertos bloques de la pagina web según criterio personal. Aprendimos a realizar espacios en blanco con (signo menor + P + signo mayor) y cerrar el parrafo con (signo menor menor + / + p + signo mayor), también podemos agregar espacio entre bloques con las letras (signo menor + b + r + signo mayor) esto dos veces por cada espacio deseado.

Sesion 2 - Agregar video

Descargar un video y ubicarlo en la carpeta de imágenes de la plantilla de la web, posteriormente puedes agregar diferentes atributos como controles al video, tamaño entre otros. Tambien creamos un enlace donde primero ubicamos el botón que dirije a la red social y posteriormente colocar el link de pagina deseada con otros atributos como por ejemplo que el link se dirija a otra pestaña.
